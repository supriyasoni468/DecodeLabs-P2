# 🌸 Iris Flower Classification using Machine Learning

## 📌 Project Overview

This project is developed as part of **DecodeLabs Artificial Intelligence Internship - Project 2: Data Classification Using AI**.

The objective of this project is to build a machine learning classification model that can accurately predict the species of an Iris flower based on its physical characteristics.

---

## 🎯 Objective

* Load and understand a dataset
* Perform data preprocessing
* Split data into training and testing datasets
* Train a classification model
* Evaluate model performance using accuracy and classification metrics

---

## 📊 Dataset

**Dataset Used:** Iris Flower Dataset

The dataset contains 150 flower samples belonging to three species:

* Iris Setosa
* Iris Versicolor
* Iris Virginica

### Features

* Sepal Length (cm)
* Sepal Width (cm)
* Petal Length (cm)
* Petal Width (cm)

### Target Variable

* Species

## ⚙️ Machine Learning Workflow

### 1. Data Loading

The Iris dataset was loaded and inspected for structure, missing values, and class distribution.

### 2. Data Preprocessing

* Removed unnecessary columns
* Encoded categorical target labels using LabelEncoder

### 3. Train-Test Split

The dataset was divided into:

* 80% Training Data
* 20% Testing Data

### 4. Feature Scaling

StandardScaler was used to normalize feature values.

### 5. Model Training

A **K-Nearest Neighbors (KNN)** classification algorithm was used to train the model.

### 6. Model Evaluation

The model performance was evaluated using:

* Accuracy Score
* Classification Report
* Confusion Matrix
